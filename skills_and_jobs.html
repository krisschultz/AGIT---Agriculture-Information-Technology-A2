Full-Stack Web Developer



Job Description

A self-motivated creative mind that is passionate in developing fully functional high-quality web applications and websites. This role demands both technical and non-technical skills. You will mainly be working on the web application that customers use to calculate feed demand and secondly, the group website. Your technical skills will be used in developing the functional aspects of these two tasks while your non-technical skills are used in developing how they are presented. You will work alongside our other developers in a collaborative effort to synchronise the project web application and mobile application in terms of both functionality and user interface design.

Key Responsibilities
  • Develop a functional web application that interacts with the project hardware
  • Design a user-friendly interface that is simple understand but also highly functional
  • Work together with both technically knowledgeable and non-technically familiar group members
  • Maintain and improve the web application and group website over time
Skills and Experiences Required
  • Experience with JavaScript and other web technologies
  • Experience with HTML5 and CSS3
  • Knowledgeable in SQL databases
  • Self-motivated and able to work independantly
  • Good communication skills
  • Able to work independently
  • Able to speak english

UI / UX Developer



Job Description

Looking for a passionate UI / UX Designer who is able to work well with others. You will work alongside our group in developing a user-friendly experience for our web application. Your mission is to develop solutions to creating a highly functional user interface while maintaining simplicity so any person can understand how to use the application. Your passion for user interface design will deliver our project idea in a presentable fashion for the regular farmer.

Key Responsibilities
  • Designing a consistent user interface across the mobile application and web application.
  • Working together with our developers to bring ideas to reality.
  • Working iteratively and collaboratively with the rest of the group to build upon feedback and improve ideas and designs.
  • Understand the needs of the users (farmers).
  • Research and understand cow feeding/ farming.
Skills and Experiences Required
  • Experience with hands-on UI / UX design.
  • Experience with creative design tools such as Adobe Photoshop, Adobe Illustrator, etc.
  • Deep knowledge UI / UX design.
  • Understanding of end user needs.
  • Good communication skills.
  • Able to work collaboratively.
  • Able to speak good English.

Mobile App Developer



Job Description

We are looking for a skilled and experienced mobile app developer who is passionate about our project idea. You will be given the task of developing the mobile application(s) that will interact with our feed calculator system. Collaborate with the team to design and implement a working platform that will allow farmers to interact with out technology. Use your technical expertise to realise functional mobile applications that can run on almost any smart phone or device.

Key Responsibilities
  • Develop a functional mobile application that allows users to interact with our feed calculator system.
  • Work with our UI/UX Designer to develop a user friendly interface for the mobile application.
  • Work with our Full-Stack Developer to integrate Front and Back-end technologies.
  • Writing clean, readable, reusable code.
  • Test and fix bugs within the program.
  • Optimise code for performance on mobile devices.
Skills and Experiences Required
  • Experience with Swift and/or Java for mobile application development.
  • Strong knowledge of mobile operating systems.
  • Experience with testing and debugging mobile applications.
  • Familiarity with mobile app standards and guidelines for various platforms.
  • Good communication skills.
  • Able to work collaboratively.
  • Able to speak good English.

Hardware Engineer



Job Description

You will be the primary designer of our integrated hardware. Using your engineering skills, you will develop a user interface with Raspberry Pis and processors and sensors with Arduinos. These two technologies will enable farmers to use our feed calculator to more efficiently plan their cow feeding. Your creativity, innovation, and technical skills will be put to the test in designing a working product.

Key Responsibilities
  • Developing a user interface that farmers can interact with.
  • Designing processors that handle information from sensors and actuators.
  • Testing and improving hardware functionality.
  • Analysing and solving technical and hardware related issues.
Skills and Experiences Required
  • Experience with Raspberry Pis and/or Arduinos.
  • Experience in software engineering.
  • Experience with physical design.
  • Able to work collaboratively.
  • Able to speak good English.
  • Passionate in testing and solving problems.
